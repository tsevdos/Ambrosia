This theme was lovingly designed and developed by John Tsevdos who blogs at http://phrappe.com and http://www.tsevdos.com.

Some of it's features :
	* Valid HTML5 markup
	* CSS queries! Theme adjust itself to desktop, iPad and mobile phones screens!
	* Widget ready, with 4 different widget areas
	* Support for two Custom menus (main and secondary), at the sidebar (the secondary menu is exactly below of the main menu)
	* Support for threaded/nested comments
	* Super light-weight
	* Sticky posts
	* Translation ready
	* Want more? accessible, super clean, SEO-friendly and many more

The theme works in Wordpress 2.7 and above (it was developed and tested in Wordpress 3). If you have any tips/issues please inform me (http://phrappe.com/contact/).
Tested and works fine with the below bowsers :
	* Chrome
	* Firefox
	* IE 9, 8 and 7
	* Opera
	* Safari
	* Safari for iPhone/iPad
